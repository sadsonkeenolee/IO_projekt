# Pliki konfiguracyjne.
Tutaj znajduja sie wszystkie pliki konfiguracyjne.
## Nazwy plikow.
Obecnie serwisy obsluguja:
- `CredentialService.toml` 
    ```toml
        [DatabaseConfig]
        name = "baza_danych"    # name of a database
        port = "3306"           # port for connection 
        username = "login"      # username for logging credentials
        password = "haslo"      # password for logging credentials
    ```
    Nalezy zapoznac sie z standardem dla danej bazy danych.
